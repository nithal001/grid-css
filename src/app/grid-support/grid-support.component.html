Grid Support







Use feature queries to test for grid support in the current browser.
Feature queries asks the browser whether it supports grid Works like media query.


Practical Approach Today

  • Build accessible mobile first layout without grid. Make it work on mobile and make sure it works on large screen.
  • Use that mobile layout as a fallback for all browsers
  • Use @support to detect grid support
  • At appropriate breakpoint create layout with grid and grid area
  • Explore new layout as viewport widens.
